Criando uma Tabela

A estrutura básica de uma tabela é:

  • <table>: O container da tabela.
  • <thead>: Cabeçalho da tabela.
  • <tbody>: Corpo da tabela com os dados.
  • <tr>: Table Row (Linha da tabela).
  • <th>: Table Header (Célula de cabeçalho, negrito por padrão).
  • <td>: Table Data (Célula de dados comuns).
